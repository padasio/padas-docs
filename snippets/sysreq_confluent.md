PADAS requires [Confluent Platform](https://www.confluent.io) in order to operate properly.  Namely, the following Confluent Platform components must be running:

- ZooKeeper
- Kafka Broker


Version Compatibility

| Padas   | Confluent Platform  |
| ------- | ------------------- |
| 0.0.x   | 6.x & 7.x           |
